## Quick start

Assuming that you will run in a local development environment, follow the instructions below:

#### Install `yarn`

This repository uses `yarn` v3 (berry).

#### Install the packages

Run `yarn` to install the node packages

### Build the packages

To build the packages execute:

```bash
yarn run build
```

## Run Storybook

Run storybook for development

```bash
yarn run storybook
```

A compiled storybook can be found also [here](https://ensign-docs.rotational.io).

## Publish to NPM

You can publish the package by running

```bash
yarn run release
```

## System Requirements

- git
- NodeJS `>=16.0.0`
- yarn v3

## Packages used

- tailwind css
- cva (class-variance-authority)
- clsx
